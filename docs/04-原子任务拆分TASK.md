# 原子任务拆分 (Atomic Tasks)

## 概述

本文档将Text-to-SQL项目拆分为200+个原子任务，每个任务都是可独立执行、明确边界的工作单元。

### 任务标记说明

| 标记 | 说明 | 示例 |
|-----|------|-----|
| `M:模块` | 所属模块 | M:Foundation, M:Database, M:AI |
| `D:优先级` | 开发优先级 | D:P0(必须), D:P1(重要), D:P2(期望) |
| `D:依赖` | 依赖任务 | D:T001 |
| `P:并行` | 可并行开发 | P:PARALLEL |
| `H:工时` | 预估工时(小时) | H:2h |

---

## 模块划分总览

```
模块分层 (共7大模块)
├── M0: Foundation (基础设施层)        ~40任务
├── M1: Database (数据库层)           ~35任务
├── M2: Query (查询执行层)            ~30任务
├── M3: AI (AI服务层)                 ~35任务
├── M4: API (接口层)                  ~30任务
├── M5: Export (导出层)               ~20任务
└── M6: DevOps (DevOps层)             ~20任务
                                       
总计: ~210 任务
```
